# intern-au

A simple site for finding internships in Australia. A work in progress.

Uses Express.js and MongoDB.

### Running

```bash
npm start
```

### Assets

```bash
cd assets && ./build
```

### Seed Database

To see the database with sample companies, run:

```bash
$ coffee scripts/seed.coffee
```
